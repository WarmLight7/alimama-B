# 使用方式
- 安装 docker 和 docker-compose
- 在 code 目录中编写代码
- code目录包含build.sh和run.sh 这2个脚本, 具体功能可以参考 demo
- ./run.sh可以读取 CPU、MEMORY、NODE_NUM、NODE_ID等环境变量，判断容器所在环境，启动不同的进程
- 执行`bash run_local_test.sh`进行测试
- 测试通过后，上传脚本生成的code.tar.gz到评测平台进行测试
